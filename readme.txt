AM - YouTube it

== Description ==

Insert YouTube&trade; videos directly in your post in one of the following ways:

1. Embed video.
2. shortCode for embed video.
3. YouTube image with popup video using thickbox.
4. YouTube image with a link to the video on YouTube website.
5. YouTube original title with a link to the video on YouTube website. 

You can insert the original title and description in your post.
You can resize the embed video or image.

"AM - YouTube it" works out of the box. Just install it, activate it and use it.

== Installation ==

1. Upload the am-youtube-it folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on new post and you will find the plugin button next to the media button.
